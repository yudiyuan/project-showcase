Personal Portfolio Website

1. Project Overview

This project is a personal portfolio website designed to introduce myself, showcase my skills, and share my career aspirations. It includes different sections such as "About Me", "Skills", "More About Me", and "Reflection & Future Plans".

2. Design Process

Before implementing the website, I first sketched a boxified layout to plan the structure and organization of elements. The design follows a responsive layout, ensuring that the website adapts to different screen sizes (desktop, tablet, mobile).

The page structure consists of a fixed navigation bar, a header section, and multiple content sections with distinct background colors to enhance readability.

3. Technologies Used

  • HTML5 - For structuring the webpage
  • CSS3 - For styling and layout customization
  • Bootstrap 5 - For responsive design and pre-built components
  • Git & GitHub - For version control and online deployment

4. GitHub Deployment

The website was deployed using GitHub Pages. The steps followed:

  1. Created a public repository on GitHub named `Assignment1`.
  2. Initialized a Git repository locally and committed all files.
  3. Pushed the code to the GitHub repository.
  4. Enabled GitHub Pages in the repository settings.
  5. Deployed successfully at: GitHub Pages Link

5. Challenges & Solutions

During the development process, I faced the following challenges:

  • Issue: Ensuring that all sections were properly styled and aligned.
  • Solution: Used CSS box model with `padding`, `margin`, and `border` for consistent spacing.
  • Issue: Background colors did not change on small screens.
  • Solution: Used a `@media` query to change section colors for mobile screens.

6. Future Improvements

In the future, I would like to enhance this project by adding:

  • JavaScript interactions (e.g., animations, dynamic content updates)
  • User feedback form to collect comments and suggestions
  • Project showcase section displaying my latest work
